# Student Grades Analysis

This project analyzes student grades using Python, NumPy, and Matplotlib.  
It calculates the average, highest, and lowest scores, counts passing and failing students,  
and visualizes grades with bar charts.

## Features

- Calculate average, highest, and lowest scores for each student.
- Count the number of passing and failing students.
- Visualize student grades with bar charts.

## How to Run

1. Open `grades_analysis.ipynb` in Jupyter Notebook.
2. Run all cells to see the analysis and charts.
